# AI Sales Copilot

Помощник менеджера по продажам электронники (но вообще можно, что угодно продавать, только надо Базу Знаний подходящую вгрузить): слушает разговор и в реальном времени
подсказывает возражения клиента, тональность, этап сделки, что ответить и какой
товар предложить.

**Стек:** FastAPI (REST + WebSocket) + фронт, БД для истории сессий.

## Как пользоваться

1. Откройте приложение в Chrome или Safari (микрофон работает на `localhost` или
   по HTTPS).
2. «Разрешить микрофон» → «Старт записи».
3. Говорите, удерживая клавишу: **Shift** — клиент, **Пробел** — менеджер.
   Отпустили — фрагмент расшифровался, справа появилась подсказка.

Тему, язык (RU/EN), профиль клиента и настройки LLM/распознавания можно менять
прямо в интерфейсе.

## Запуск локально

Ключ к LLM уже встроен, после клона всё работает сразу.

**Docker:**

```bash
git clone https://github.com/Janko025/ai-sales-copilot.git
cd ai-sales-copilot
docker compose up -d --build   # приложение + PostgreSQL
```

Откройте http://localhost:5000.

**Без Docker** (нужны Python 3.10+ и `ffmpeg` в PATH):

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python app.py        # http://127.0.0.1:5000
```
